# amr-vivli-2026

Research repository for the 2026 AMR Vivli Challenge: data processing pipelines, analytic methodology, Section 7 deliverables, and reproducible research artifacts.

## Pipeline

All work lives under [`preprocessing_pipeline/`](preprocessing_pipeline/README.md):

1. **Section 5 preprocessing** — `python preprocessing_pipeline/run_pipeline.py` (Steps 1–10 → master table + crosswalks)
2. **Section 6 analytics** — `step11` through `step17` (descriptive profiling through intervention impact)
3. **Section 7 deliverables** — `step18_section7_deliverables.py` → [`preprocessing_pipeline/deliverables/`](preprocessing_pipeline/deliverables/README.md)

Project brief and methodology plans: `preprocessing_pipeline/docs/`.
